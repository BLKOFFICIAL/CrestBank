# 🏦 CrestBank

CrestBank is a modern banking system for Minecraft servers that introduces secure, GUI-based financial management to Vault-powered economies.

Players can safely store money, transfer funds, earn passive interest, and trade physical bank notes, while administrators gain powerful tools to monitor and protect the server economy.

Designed for **Minecraft 1.21.1+**, CrestBank bridges the gap between simple wallet money and a fully managed banking system.

---

# 🚀 Features

## 🖥️ Interactive Banking GUI
CrestBank replaces long commands with a simple and intuitive inventory interface.

- Clean **Deposit** and **Withdraw** sections
- Quick actions for **25%, 50%, 75%, and 100%** transactions
- Displays both **Bank Balance** and **Wallet Balance** in real time
- Easy one-click transfers between wallet and bank

![Bank GUI](https://i.ibb.co/Jws2CJCc/Screenshot-2026-03-10-170242.png)

---

## 📈 Passive Interest System
Players earn passive income by storing money in the bank.

- Automatic interest payouts
- Optional server events using **interest boosts**
- Clear notifications when interest is received

![Interest Messages](https://i.ibb.co/vb8bJsJ/Screenshot-2026-03-10-170614.png)

---

## 💵 Physical Bank Notes
CrestBank allows players to convert digital balance into physical bank notes that can be traded with other players.

- Create notes using `/bank note <amount>`
- Perfect for player trading and economy gameplay

---

## 🔒 Anti-Duplication Protection
Bank notes include **unique identifiers** to prevent duplication exploits.

If a duplicated or modified note is detected, CrestBank automatically rejects it to protect the server economy.

![Invalid Bank Note](https://i.ibb.co/393pdMTj/Screenshot-2026-03-10-170847.png)

---

# 🛡️ Administrative Controls

CrestBank includes powerful moderation and economy protection tools.

- **Maintenance Mode** – temporarily disable banking interactions
- **Global Lockdown** – freeze all transactions instantly
- **Account Freezing** – block suspicious players from using the bank
- **Transaction Logs** – every transaction is recorded for auditing
- **Audit Tools** – investigate player financial activity
- **Rate Limits & Caps** – protect the economy from abuse

---

# ⚙️ Requirements

- **Minecraft:** 1.21.1+
- **Java:** 17+
- **Vault:** required for economy integration
- **MySQL / MariaDB:** recommended for best performance
